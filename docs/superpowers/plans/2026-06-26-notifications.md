# Sub-project #12 — Notifications (in-app center + email/webhook channels) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Deliver a persistent, per-user notification center — a bell icon with live unread count, a notification list page, per-channel delivery preferences, email delivery via Laravel's mail stack, and a custom webhook/Slack channel. Feed the first two live event sources (`operation.finished` and `ssl.issued`), stub the rest for future sub-projects, and wire the bell count to Reverb so it updates without polling.

**Architecture:** Event source → `NotificationService::dispatch($user, $notification)` → preference filter (`NotificationPreference::isEnabled`) → `$user->notify()` → resolved channels (database / mail / `WebhookChannel`). `DatabaseChannel` write triggers `NotificationsObserver::created()` → broadcasts `NotificationCreated` on `private-notifications.{userId}`. `useNotifications()` React hook subscribes and maintains `unreadCount`; `NotificationBell` shows the badge live.

**Tech Stack:** Laravel 12 (Notifications, Mail, Events, Observers), Pest 3, Reverb (Echo/pusher-js), Inertia + React (JSX), `Http` facade, MySQL (prod) / SQLite `:memory:` (tests).

**Design spec:** `docs/superpowers/specs/2026-06-26-notifications-design.md`

## Global Constraints

- **Broadcast channel is exactly `notifications.{userId}`**; event `broadcastAs` name is exactly `NotificationCreated`; payload carries only `{ unread_count: int }`.
- **`notification_preferences` opt-out model:** missing row = enabled. `isEnabled()` returns `true` on any DB error (fail-open).
- **`WebhookChannel` is non-critical:** catch all `\Throwable`, log with `Log::warning`, never rethrow.
- **`ShouldBroadcastNow`** (not `ShouldBroadcast`) for `NotificationCreated` — skips queue for immediate bell update.
- **`OperationJob` hook is opt-in:** `protected ?User $notifyUser = null`; base `handle()` calls `NotificationService::dispatch()` only when not null.
- **`notifications` table** created by `php artisan notifications:table` — no custom columns. User already has `Notifiable` trait (confirmed `app/Models/User.php` line 17).
- **Channel auth:** `notifications.{userId}` scoped to own user ONLY — no admin cross-view (unlike `operations.{userId}`).
- **Tests run with `QUEUE_CONNECTION=sync`, `Event::fake()`, `Mail::fake()`, `Http::fake()`.**
- **Branch:** `feature/notifications` (off `main`). Each task commits here.
- **Run the suite in the `local-dev` container** for the authoritative result: `docker exec laranode-lab bash -lc 'cd /home/laranode_ln/panel && php artisan test'`. On Windows, use PowerShell for `make`/`docker compose`; plain `docker exec laranode-lab …` works from any shell.

<?php // app/Models/NotificationPreference.php

namespace App\Models;

use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\BelongsTo;

class NotificationPreference extends Model
{
    protected $fillable = ['user_id', 'event_type', 'channel', 'enabled'];

    protected $casts = ['enabled' => 'boolean'];

    public function user(): BelongsTo
    {
        return $this->belongsTo(User::class);
    }

    /**
     * Returns true when the channel is enabled for the given user + event type.
     * Missing row = enabled (opt-out model). Fail-open on DB error.
     */
    public static function isEnabled(int $userId, string $eventType, string $channel): bool
    {
        try {
            $pref = static::where('user_id', $userId)
                ->where('event_type', $eventType)
                ->where('channel', $channel)
                ->first();

            return $pref === null || $pref->enabled;
        } catch (\Throwable) {
            return true; // fail-open: deliver rather than silently suppress
        }
    }
}
```

<?php // app/Services/Notifications/NotificationService.php

namespace App\Services\Notifications;

use App\Models\NotificationPreference;
use App\Models\User;
use App\Notifications\Channels\WebhookChannel;
use Illuminate\Notifications\Notification;

class NotificationService
{
    /**
     * Resolve which channels are enabled for a given user + event type.
     * Returns channel driver names / class names that Laravel's Notification system understands.
     */
    public static function resolveChannels(object $notifiable, string $eventType): array
    {
        $available = ['database', 'mail', WebhookChannel::class];

        return array_values(array_filter(
            $available,
            fn ($channel) => NotificationPreference::isEnabled($notifiable->id, $eventType, $channel)
        ));
    }

    /**
     * Dispatch a notification with preference filtering applied.
     * Call this from every event source instead of $user->notify() directly.
     */
    public static function dispatch(User $user, Notification $notification): void
    {
        $user->notify($notification);
    }
}
```

<?php // app/Notifications/Channels/WebhookChannel.php

namespace App\Notifications\Channels;

use Illuminate\Notifications\Notification;
use Illuminate\Support\Facades\Http;
use Illuminate\Support\Facades\Log;

class WebhookChannel
{
    public function send(object $notifiable, Notification $notification): void
    {
        if (empty($notifiable->webhook_url)) {
            return;
        }

        try {
            Http::timeout(10)->post($notifiable->webhook_url, $notification->toWebhook($notifiable));
        } catch (\Throwable $e) {
            Log::warning('WebhookChannel delivery failed', [
                'url' => $notifiable->webhook_url,
                'error' => $e->getMessage(),
            ]);
        }
    }
}

<?php // app/Notifications/OperationFinishedNotification.php

namespace App\Notifications;

use App\Models\Operation;
use App\Services\Notifications\NotificationService;
use Illuminate\Bus\Queueable;
use Illuminate\Contracts\Queue\ShouldQueue;
use Illuminate\Notifications\Messages\MailMessage;
use Illuminate\Notifications\Notification;

class OperationFinishedNotification extends Notification implements ShouldQueue
{
    use Queueable;

    public function __construct(public Operation $operation) {}

    public function via(object $notifiable): array
    {
        $eventType = $this->operation->status === 'failed' ? 'operation.failed' : 'operation.finished';
        return NotificationService::resolveChannels($notifiable, $eventType);
    }

    public function toDatabase(object $notifiable): array
    {
        return [
            'event_type' => $this->operation->status === 'failed' ? 'operation.failed' : 'operation.finished',
            'operation_id' => $this->operation->id,
            'type' => $this->operation->type,
            'target' => $this->operation->target,
            'status' => $this->operation->status,
        ];
    }

    public function toMail(object $notifiable): MailMessage
    {
        $label = $this->operation->status === 'failed' ? 'failed' : 'finished';
        return (new MailMessage)
            ->subject("Operation {$label}: {$this->operation->type}")
            ->line("Your operation \"{$this->operation->type}\" on \"{$this->operation->target}\" has {$label}.")
            ->line("Status: {$this->operation->status}")
            ->action('View Operations', url('/admin/operations'));
    }

    public function toWebhook(object $notifiable): array
    {
        return [
            'event_type' => $this->operation->status === 'failed' ? 'operation.failed' : 'operation.finished',
            'operation_id' => $this->operation->id,
            'operation_type' => $this->operation->type,
            'target' => $this->operation->target,
            'status' => $this->operation->status,
        ];
    }
}

        $this->operation->markRunning();

        try {
            $exit = $this->run(fn (string $line) => $this->operation->appendOutput($line));

            NotificationService::dispatch(
                $this->notifyUser,
                new OperationFinishedNotification($this->operation)
            );
        }
    }

<?php // app/Http/Controllers/NotificationPreferencesController.php

namespace App\Http\Controllers;

use App\Models\NotificationPreference;
use App\Notifications\Channels\WebhookChannel;
use Illuminate\Http\JsonResponse;
use Illuminate\Http\Request;
use Inertia\Inertia;
use Inertia\Response;

class NotificationPreferencesController extends Controller
{
    /** Known event types. Extend as new sources ship. */
    private const EVENT_TYPES = [
        'operation.finished',
        'operation.failed',
        'ssl.issued',
        'ssl.expiring',
        'fail2ban.ban',
        'resource.threshold',
        'backup.result',
        'deploy.success',
        'deploy.failed',
    ];

    private const CHANNELS = ['database', 'mail', WebhookChannel::class];

    public function index(Request $request): Response
    {
        $userId = $request->user()->id;
        $rows = NotificationPreference::where('user_id', $userId)->get()
            ->keyBy(fn ($p) => "{$p->event_type}:{$p->channel}");

        $preferences = [];
        foreach (self::EVENT_TYPES as $eventType) {
            foreach (self::CHANNELS as $channel) {
                $key = "{$eventType}:{$channel}";
                $preferences[$key] = [
                    'event_type' => $eventType,
                    'channel' => $channel,
                    'enabled' => isset($rows[$key]) ? $rows[$key]->enabled : true,
                ];
            }
        }

        return Inertia::render('Profile/Notifications', [
            'eventTypes' => self::EVENT_TYPES,
            'channels' => self::CHANNELS,
            'preferences' => array_values($preferences),
        ]);
    }

    public function update(Request $request): JsonResponse
    {
        $validated = $request->validate([
            'event_type' => ['required', 'string', 'in:' . implode(',', self::EVENT_TYPES)],
            'channel' => ['required', 'string', 'in:' . implode(',', self::CHANNELS)],
            'enabled' => ['required', 'boolean'],
        ]);

        NotificationPreference::updateOrCreate(
            [
                'user_id' => $request->user()->id,
                'event_type' => $validated['event_type'],
                'channel' => $validated['channel'],
            ],
            ['enabled' => $validated['enabled']]
        );

        return response()->json(['ok' => true]);
    }
}

```js
// resources/js/hooks/useNotifications.js
import { useEffect, useState } from 'react';
import { usePage } from '@inertiajs/react';

export default function useNotifications() {
    const { auth, notifications } = usePage().props;
    const userId = auth.user?.id;
    const [unreadCount, setUnreadCount] = useState(notifications?.unreadCount ?? 0);

    useEffect(() => {
        if (!userId) return;

        const channel = window.Echo.private(`notifications.${userId}`);
        channel.listen('.NotificationCreated', (e) => {
            setUnreadCount(e.unread_count);
        });

        return () => channel.stopListening('.NotificationCreated');
    }, [userId]);

    const refresh = () => setUnreadCount(0); // caller can reset after mark-all-read

    return { unreadCount, refresh };
}
```

// resources/js/Pages/Profile/Notifications.jsx
import AuthenticatedLayout from '@/Layouts/AuthenticatedLayout';
import { Head, usePage } from '@inertiajs/react';
import { useState } from 'react';
import axios from 'axios';
import { ImProfile } from 'react-icons/im';

const channelLabel = {
    'database': 'In-app',
    'mail': 'Email',
    'App\\Notifications\\Channels\\WebhookChannel': 'Webhook',
};

export default function Notifications({ eventTypes, channels, preferences: initialPrefs }) {
    const { auth } = usePage().props;
    const [prefs, setPrefs] = useState(() => {
        const map = {};
        initialPrefs.forEach((p) => { map[`${p.event_type}:${p.channel}`] = p.enabled; });
        return map;
    });
    const [webhookUrl, setWebhookUrl] = useState(auth.user?.webhook_url ?? '');
    const [saving, setSaving] = useState(false);

    const toggle = (eventType, channel, current) => {
        const key = `${eventType}:${channel}`;
        const newValue = !current;
        setPrefs((prev) => ({ ...prev, [key]: newValue }));
        axios.patch('/profile/notifications', { event_type: eventType, channel, enabled: newValue });
    };

    const saveWebhook = () => {
        setSaving(true);
        axios.patch('/profile', { webhook_url: webhookUrl })
            .finally(() => setSaving(false));
    };

    return (
        <AuthenticatedLayout header={
            <h2 className="text-xl font-semibold flex items-center">
                <ImProfile className="mr-2" /> Notification Preferences
            </h2>
        }>
            <Head title="Notification Preferences" />
            <div className="pb-12 pt-8 max-w-4xl px-4 space-y-6">
                <div className="bg-white p-4 shadow sm:rounded-lg sm:p-8 dark:bg-gray-950">
                    <h3 className="text-lg font-medium mb-4">Delivery Channels</h3>
                    <table className="w-full text-sm">
                        <thead>
                            <tr className="text-left border-b">
                                <th className="py-2 font-medium">Event</th>
                                {channels.map((ch) => (
                                    <th key={ch} className="py-2 font-medium text-center">{channelLabel[ch] ?? ch}</th>
                                ))}
                            </tr>
                        </thead>
                        <tbody>
                            {eventTypes.map((et) => (
                                <tr key={et} className="border-b">
                                    <td className="py-2 font-mono text-xs">{et}</td>
                                    {channels.map((ch) => {
                                        const key = `${et}:${ch}`;
                                        const enabled = prefs[key] !== undefined ? prefs[key] : true;
                                        return (
                                            <td key={ch} className="text-center py-2">
                                                <input
                                                    type="checkbox"
                                                    checked={enabled}
                                                    onChange={() => toggle(et, ch, enabled)}
                                                    className="rounded"
                                                />
                                            </td>
                                        );
                                    })}
                                </tr>
                            ))}
                        </tbody>
                    </table>
                </div>

                <div className="bg-white p-4 shadow sm:rounded-lg sm:p-8 dark:bg-gray-950">
                    <h3 className="text-lg font-medium mb-2">Webhook / Slack URL</h3>

                        >
                            {saving ? 'Saving…' : 'Save'}
                        </button>
                    </div>
                </div>

    );
}
